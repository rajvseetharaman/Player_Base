About the System

Author: Rajendran Seetharaman

This is a cricket player info system which gives the following information about a cricket player

  1. The demographic information about the player like nationality, playing role, batting, and bowling styles
  2. The match statistics for the player like number of matches playes, runs scored, and wickets taken.

My motive behind developing this system was that it could be used by avid cricket followers to look up information about their favorite player organized in a useful manner.
